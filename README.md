# 📊 Stock Dashboard

Live inventory dashboard with KPIs, alerts, and charts.

## Features
- Upload Excel/CSV files
- View Key Performance Indicators (KPIs)
- See critical alerts for low stock items
- Interactive charts and graphs
- Search and filter data
- Export data as CSV

## How to Use
1. Upload your Excel or CSV file
2. View the dashboard automatically
3. Check KPIs and alerts
4. View charts and data table
5. Export data if needed

## Supported File Formats
- Excel (.xlsx, .xlsm, .xls)
- CSV (.csv)

## Expected Column Names
Your file should have these columns:
- ITEM or ITEM NAME
- CODE
- SAFETY STOCK
- OPENING STOCK
- CLOSING STOCK
- ISSUED
- RECEIVED
